#랜더링 파이프라인이란?
 3D 좌표라는 숫자데이터가 여러 공정을 거쳐 모니터의 픽셀(색상 점)로 변하는 일련의 제조 공정


##3d최적화 방법
- 안그린다
- 눈속임을 한다.


###로컬 스페이스 : 나 중심
- 물체 자신이 주인공인 공간
- 중심점 : 물체의 정중앙

###월드 스페이스 : 세상 중심
- 게임 속 가상 세계를 기준으로 한 공간
- 중심점 : 게임 맵의 절대적인 원점(0,0,0)

###뷰스페이스 : 눈(카메라)중심
- 관찰자를 기준으로 세상을 재정렬한 공간
- 중심점 : 내 카메라 렌즈의 위치


## 컬링
### Back-face Culling
- 상자의 앞면은 보이지만 뒷면은 절대 안보이죠? GPU가 이 뒷면을 그리지 않게 만드는 설정
- 시계 방향(CW) : 앞면 -> 그린다
- 반시계 반향(CCW) : 뒷면 -> 안그린다

###View Frustum Culling
- 시야 범위 밖에 있는 물체들을 통째로 버림

###Occlusion Culling
- 가려져서 안보이는거 안그림

##클리핑

- 화면 경계선에 걸친 삼각형만 가위로 슥삭 잘라버림

##투영
- 3D 입체를 2D 평면으로 찍어 누르면 원근감을 줌
